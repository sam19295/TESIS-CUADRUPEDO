En esta sección se presentan los archivos utilizados para el diseño y la fabricación del chasis completo del robot cuadrúpedo. 

El chasis fue desarrollado a partir de piezas modulares y fácilmente modificables, lo que permite realizar ajustes o mejoras sin necesidad de rediseñar toda la estructura. 

Esta característica facilita el ensamblaje, mantenimiento y adaptación del robot para futuras iteraciones o implementaciones con distintos componentes electrónicos y mecánicos.
